# FastHTTP Examples

All examples are tested and working.

## Categories

- **basic** — basic HTTP methods (GET, POST, PUT, DELETE, PATCH)
- **headers** — working with headers
- **parameters** — query parameters
- **data** — sending form data
- **endpoints** — working with various APIs
- **advanced** — redirects and other advanced features
- **error** — automatic error handling and testing various error scenarios
- **middleware** - test middleware for FastHTTP
- **pydantic** - pydantic response validation examples

## Tested APIs

- [httpbin.org](https://httpbin.org)
- [jsonplaceholder.typicode.com](https://jsonplaceholder.typicode.com)
- [ipify.org](https://ipify.org)
- [httpstat.us](https://httpstat.us)
