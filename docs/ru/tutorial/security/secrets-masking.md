# Маскирование секретов

Автоматическое скрытие чувствительных данных в логах.

## Как это работает

При включенном режиме отладки FastHTTP автоматически маскирует чувствительные данные:

### Маскируемые заголовки

- `Authorization`
- `Cookie`
- `X-API-Key`

### Маскируемые параметры URL

- `api_key`
- `token`
- `password`

## Пример

Вместо:

```
Authorization: Bearer sk-1234567890abcdef
```

В логах:

```
Authorization: *****
```

## Включение режима отладки

```python
app = FastHTTP(debug=True)
```

## Отключение

```python
app = FastHTTP(security=False)
```

Не рекомендуется для продакшена.
