# HTTP/2

Включение HTTP/2 для улучшенной производительности.

## Установка

Для поддержки HTTP/2 установите дополнительные зависимости:

```bash
pip install fasthttp-client[http2)
```

## Включение HTTP/2

```python
from fasthttp import FastHTTP

app = FastHTTP(http2=True)
```

## Что такое HTTP/2?

HTTP/2 - это вторая версия протокола HTTP с возможностями:

- **Мультиплексирование** - несколько запросов через одно соединение
- **Сжатие заголовков** - меньше данных передается
- **Server Push** - сервер отправляет данные заранее
- **Приоритизация** - важные ресурсы загружаются первыми

## Производительность

### До HTTP/2

```
Соединение 1: GET /page
Соединение 2: GET /style.css
```

### С HTTP/2

```
Соединение 1: GET /page, /style.css (параллельно)
```

## Ограничения

- Не все серверы поддерживают HTTP/2
- Требует HTTPS
- Не поддерживается в некоторых прокси
