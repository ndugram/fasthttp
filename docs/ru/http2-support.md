# HTTP/2

Включите HTTP/2 для лучшей производительности.

## Установка

```bash
pip install fasthttp-client[http2]
```

## Включение

```python
app = FastHTTP(http2=True)
```

## Возможности

- Мультиплексирование — несколько запросов на одном соединении
- Сжатие заголовков (HPACK)
- Меньшая задержка

## Требования

- HTTPS обязательно
- Сервер должен поддерживать HTTP/2

## Проверка

```python
print(app._http2)  # True или False
```

Автоматически переключается на HTTP/1.1 если сервер не поддерживает HTTP/2.
